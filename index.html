A bit about Web Servers...

In short, a web server is a software service that can accept requests and return responses. This is how we access information online, we make resquests and the software behind the application returns information based on that request. An important part of organizing web servers is the Domain System (DNS) which allows user to find what they are looking for with URLs. Registering your applications Domnain Name and IP address (server location) alows users to easily find your website. While any Domain name is possible, most domains have already been purchased, and there is a significant market for re-selling these domain names. This is a breif overview of Web Servers and the DNS.
